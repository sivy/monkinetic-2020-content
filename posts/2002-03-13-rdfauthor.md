Via [Dan Brickley](http://rdfweb.org/people/danbri/)'s <a rhef="http://rdfweb.org/dev.html">RDFWeb</a> (dev page), I found a link to this most-amazing tool for creating RDF files. [RDFAuthor](http://rdfweb.org/people/damian/RDFAuthor/) is a Mac OS X application that loads RDF vocabularies and lets you graphically create instances of those schema. I used it to create my new [FOAF](http://xmlns.com/foaf/0.1/)-enabled [about page](http://media.redmonk.net/aboutMeRDF.txt).
